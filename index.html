li[class^="a"] выбирает все значения атрибута, которые начинаются с a

li[class$="a"] выбирает все значения атрибута, которые заканчиваются на a

li[class*="a"] выбирает все значения атрибута, где появляется a

} p { Используя li[class], мы можем выбрать каждый селектор с атрибутом класса. li[class="a"] выбирает селектор с классом a, но не селектор с классом a в сочетании с другим, отделённым запятой, классом как частью значения. li[class~="a"] выберет класс a, а также значение, которое содержит класс a как часть разделённого пробелом списка. } псевдо класы
